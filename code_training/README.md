# Code Training

Dieser Bereich zeigt kontinuierliches technisches Training.
Nicht nur fertige Ergebnisse sind wichtig, sondern auch sichtbare Uebung,
saubere Wiederholung und dokumentiertes Lernen.

## Fokus

- Algorithmen und Problemloesung
- Backend-Verstaendnis
- Frontend-Handwerk
- Python fuer Automatisierung und Analyse
- SQL und Datenlogik

## Empfehlung fuer den Aufbau einzelner Aufgaben

Jede Uebung kann enthalten:

- kurze Problemstellung
- Loesungsansatz
- finalen Code
- Learnings in 2 bis 4 Saetzen

## Ordner

- `algorithms/` fuer klassische Coding-Aufgaben
- `backend/` fuer APIs, Auth, Datenfluss und Services
- `frontend/` fuer Komponenten, States und UX
- `python/` fuer Skripte, Parsing, Automatisierung
- `sql/` fuer Queries, Joins, Reports und Datenmodelle
