# Lernen

Dieser Bereich ist fuer Tutorials, Kurse und Lernpfade gedacht.
Hier kannst du alles sammeln, was du im Rahmen von strukturiertem Lernen bearbeitest.

## Was hier hineinpasst

- Online-Kurse
- Tutorials
- Intensivprogramme
- Mitschriften
- Begleitcode zu Lerninhalten

## Struktur

- `tutorials/` fuer einzelne Tutorials und Guides
- `training/` fuer Trainingsreihen, Kurse und Lernprogramme

## Gute Regel

Wenn etwas Teil eines Kurses oder Tutorials ist, kommt es in `Lernen/`.
