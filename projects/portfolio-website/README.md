# Project: Portfolio Website

## Goal

Eine persoenliche Website, die Profil, Projekte, Skills und Kontaktmoeglichkeiten professionell praesentiert.

## Why It Matters

Dieses Projekt zeigt:

- Frontend-Qualitaet
- visuelles Verstaendnis
- persoenliches Branding

## Suggested Stack

- React oder Next.js
- TypeScript
- moderne CSS-Strategie
- Deployment ueber Vercel oder Netlify

## Core Features

- Hero Section mit klarer Positionierung
- Projektuebersicht
- About-Bereich
- Skills-Section
- Kontaktmoeglichkeit

## Nice Extras

- Dark/Light Toggle
- Projektfilter
- kleine Animationen
- Download fuer Lebenslauf
