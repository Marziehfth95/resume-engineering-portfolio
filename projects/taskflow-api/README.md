# Project: TaskFlow API

## Goal

Eine kleine, saubere Backend-Anwendung fuer Task-Management mit Auth,
CRUD-Funktionen und gut dokumentierten Endpunkten.

## Why It Matters

Dieses Projekt zeigt:

- API-Design
- Datenmodellierung
- Struktur im Backend
- Testing und Fehlerbehandlung

## Suggested Stack

- Node.js mit Express oder Fastify
- PostgreSQL
- Prisma oder SQL
- JWT Auth
- Tests mit Vitest oder Jest

## Core Features

- Benutzer registrieren und einloggen
- Tasks erstellen, aktualisieren, loeschen
- Status und Prioritaet verwalten
- Filter und Sortierung
- saubere Fehlerantworten

## Strong Resume Angle

Ein gutes API-Projekt beweist, dass du echte Backend-Grundlagen beherrschst
und nicht nur Tutorial-Code nachbaust.
