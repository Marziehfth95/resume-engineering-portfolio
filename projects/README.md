# Complete Projects

Dieser Bereich zeigt eigenstaendige Projekte mit echtem Showcase-Charakter.
Der Fokus liegt nicht nur auf Code, sondern auf Problemverstaendnis,
Produktdenken und technischer Umsetzung.

## Was ein starkes Projekt hier zeigen sollte

- klare Idee
- realistischen Nutzen
- saubere technische Entscheidungen
- gute Dokumentation
- sichtbare Weiterentwicklung

## Projektportfolio in diesem Repository

- `portfolio-website/` fuer Personal Branding und Praesentation
- `taskflow-api/` fuer Backend-, API- und Datenmodell-Kompetenz
- `analytics-dashboard/` fuer Datenvisualisierung und Produktdenken
